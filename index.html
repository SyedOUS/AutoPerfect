Getting Started

Installation Guide

Install AutoPerfect 2.0 into Adobe Illustrator in under 2 minutes. Works on Mac and Windows. No plugins, no internet required.

One-time Setup Mac & Windows Illustrator 2024–2026
Before you start: Make sure Adobe Illustrator is closed before copying the scripts folder. You only need to do this once — the scripts will then be available in every document.
macOS
Windows

macOS Installation

Tested on macOS Ventura, Sonoma & Sequoia

1

Download & Unzip

Click Download All Scripts in the sidebar. Double-click AutoPerfect_2_0.zip to extract it. You will see a folder called AutoPerfect 2.0.

2

Copy the Entire Folder

Copy the AutoPerfect 2.0 folder whole — do not rename it or extract individual scripts out of it.

3

Navigate to the Scripts Folder

Open Finder and navigate to:

Applications›Adobe Illustrator 2026›Presets›en_US›Scripts

Tip: Right-click Adobe Illustrator in Applications → Show Package Contents, then navigate inside.

4

Paste the Folder Inside Scripts

Paste AutoPerfect 2.0 inside the Scripts directory. Final structure:

Scripts›AutoPerfect 2.0›brixSmartRenditions.jsx
Scripts›AutoPerfect 2.0›MagicFix.jsx
Scripts›AutoPerfect 2.0›brix_QC_Unlocked.jsx
Scripts›AutoPerfect 2.0›text_cleanup.jsx
Scripts›AutoPerfect 2.0›exportPng.jsx
5

Open or Restart Illustrator

Launch Adobe Illustrator (or fully quit and reopen if it was already running). Illustrator scans the Scripts folder on every startup.

6

Launch Any Script

Open a .ai document, then go to:

File›Scripts›AutoPerfect 2.0›[Script Name]

All five scripts appear as a submenu under AutoPerfect 2.0.

Script 01

Smart Renditions

Generates SM, L, and XL artboard renditions proportionally scaled from the master MD artboard — cleaned, snapped, and cropped automatically.

brixSmartRenditions.jsxNon-destructiveAuto-scale4 Artboards
In one sentence: Duplicates your MD master to SM, L, and XL artboards at the correct proportional scale — with font rounding, stroke snapping, and tight artboard cropping applied automatically.
How to run

Open the Smart App Panel → click QR

Launch the Brix Smart App panel first (File → Scripts → AutoPerfect 2.0 → brix_smart_app), then click the QR button. No dialog — the script runs immediately and silently. All three target artboards will contain scaled versions of your MD artwork when it finishes.

What it does
01

Setup & Validation

Checks that artboards SM, MD, L, XL all exist. Creates any missing layers. Stops if an artboard is missing.

02

Empty Text Frame Cleanup

Removes any text frames that are blank or contain only whitespace across the entire document.

03

MD Master Cleanup

Removes double-spaces, trims leading/trailing spaces, rounds font sizes to whole numbers (min 10pt), snaps stroke widths.

04

Move & Crop MD

Shifts all MD content flush to the artboard top and trims artboard height to fit the content exactly.

05

Generate SM / L / XL

Duplicates MD content to each target layer. Scales by width ratio. Re-rounds fonts (SM: 8pt min, L/XL: 12pt min) and re-snaps strokes after scaling.

06

Align & Crop Each Rendition

Centers content horizontally, aligns to artboard top, trims artboard height to content — ready for export.

Font minimums per artboard
ArtboardMinimum Font Size
SM8 pt
MD (master)10 pt
L12 pt
XL12 pt
Key capabilities

Proportional Scaling

Uses each artboard's width-to-MD-width ratio. No distortion, no guessing.

Font Rounding

Rounds every character's font size to a whole number after scaling, enforcing per-artboard minimums.

Stroke Snapping

Cleans sub-pixel stroke widths to 0.25, 0.5, 0.75, or whole integers after scaling.

Auto Artboard Crop

Trims artboard height to the exact content boundary on all four boards — ready for export.

Script 02

Magic Fix

A comprehensive in-place cleanup that fixes every artboard — fonts, strokes, spacing, content alignment, and empty layer removal — in a single run.

MagicFix.jsxAll ArtboardsIn-place CleanupNo Duplication
Key difference from Smart Renditions: Magic Fix cleans all existing artboards in place. It does not duplicate or scale content — it is a cleanup-only tool, safe to run at any stage.
How to run

Open the Smart App Panel → click FIX

Launch the Brix Smart App panel first (File → Scripts → AutoPerfect 2.0 → brix_smart_app), then click the FIX button. A confirmation dialog appears — click OK. A summary alert shows at the end listing any removed empty layers.

What it does
01

Enable Scale Corners

Turns on Illustrator's Scale Corners preference so rounded corners behave correctly on resized items.

02

Global Empty Text Frame Removal

Deletes blank or whitespace-only text frames across the entire document. Skips locked and hidden layers.

03

Text Spacing Cleanup (per artboard)

Removes consecutive double-spaces and trims leading/trailing spaces character-by-character, preserving all mixed font style runs.

04

Font Size Enforcement (per artboard)

Custom rounding rule — fractions below 0.6 round down, 0.6+ round up. Enforces per-artboard minimums. Only writes values when a change is needed.

05

Stroke Width Snapping (per artboard)

Snaps all strokes to 0.25, 0.5, 0.75, or the nearest whole number.

06

Second Empty Frame Pass

Re-checks for text frames that became empty after the space cleanup step and removes them.

07

Move Content to Top & Crop

Shifts all content flush to the artboard top, then resizes the artboard bottom to fit content height exactly.

08

Remove Empty Layers

Recursively checks every layer and removes completely empty ones. Their names are listed in the final alert.

How to run

Open the Smart App Panel → click TXT

Launch the Brix Smart App panel first (File → Scripts → AutoPerfect 2.0 → brix_smart_app), then click the TXT button. A dialog appears with three tracking options — click your choice. A summary alert shows how many frames were processed.

What gets reset on every character

Tracking

Sets letter-spacing to your chosen value on every character individually.

Scale Reset

Both horizontal and vertical character scale restored to exactly 100%.

Baseline Shift

Any baseline offset zeroed out across all characters.

Character Rotation

Individual character rotation reset to 0° — removes any per-character tilt.

Kerning → Auto

Forces AutoKernType.AUTO on every character — removes all manual or optical overrides.

Important notes
  • Processes every text frame on every layer — no per-artboard filter.
  • Font size, family, color, and weight are not changed — only the 5 attributes above.
  • Mixed-style runs are preserved — operates character-by-character.
  • Performs two redraws to ensure Illustrator commits all changes visually.

Best Practices

Tips & Best Practices

Get the most out of AutoPerfect with these workflow tips, dos and don'ts, and reference tables for strokes and swatches.

Workflow dos and don'ts

Do

  • Name artboards exactly SM, MD, L, XL in uppercase
  • Design all artwork on the MD artboard only
  • Keep each artboard on its own matching layer
  • Save your file before running any script
  • Back up your .ai file the first time you run
  • Run Magic Fix before Smart Renditions

Don't

  • Mix case in artboard names ("sm" or "Sm" will fail)
  • Place content manually on SM, L, or XL layers — Smart Renditions will overwrite them
  • Run Smart Renditions twice — it reads MD as source every time
  • Rename the .jsx files — the File menu uses the filename as the display name
  • Run Export PNG on an unsaved document

Pro Tips

  • Run QC Report after Smart Renditions to catch any scaling artefacts
  • Run Text Cleanup before Smart Renditions if your type has inconsistent tracking
  • Magic Fix is safe to run multiple times — it won't double-process clean documents
  • Assign scripts to Illustrator Actions with keyboard shortcuts for faster access
  • The QC report is saved as HTML — open it anytime from the same folder as your .ai

Performance tips
  • Close other open documents before running Smart Renditions on complex files to free up memory.
  • Embed linked images before Export PNG to avoid broken links in the output files.
  • For large files (100+ items per artboard), allow 10–30 seconds for Smart Renditions — it processes every character individually.
  • The QC Report script is read-only and fast — safe to run as many times as needed during a project.
Support

Troubleshooting

Solutions to the most common issues encountered when installing or running AutoPerfect scripts.

Installation issues

Scripts don't appear under File → Scripts

The most common cause is that Illustrator wasn't fully restarted. Fully quit Illustrator (Cmd+Q on Mac / Alt+F4 on Windows) and reopen it. Also verify the AutoPerfect 2.0 folder is directly inside Scripts, not inside a nested subfolder.

macOS security warning blocks the script

Go to System Settings → Privacy & Security. Scroll to the Security section — click Allow Anyway next to the blocked file. Then run the script again and click Open.

Windows asks for administrator permission

This is normal for the Program Files directory. Click Continue. If you don't have admin rights, ask your IT administrator or run Illustrator as Administrator.

Runtime issues

Smart Renditions stops without doing anything

The script requires all four artboards named exactly SM, MD, L, and XL in uppercase. Check the Artboards panel and rename any that are different (e.g. "sm", "Artboard 1").

Export PNG says "Please save the document first"

Use File → Save (Cmd/Ctrl+S) then run the script again. The export script needs a saved file to determine the output folder.

QC Report doesn't open in the browser

The HTML file is still saved — look for DocumentName_QC_Report.html in the same folder as your .ai file and double-click to open it manually.

Text Cleanup dialog disappears immediately

This is an ExtendScript timing issue on some systems. Click elsewhere in Illustrator to ensure focus is on the app, then run the script again.

Magic Fix is very slow on a large document

Magic Fix operates character-by-character. Large files with many text frames may take 30–60 seconds. Do not click or interrupt Illustrator — wait for the completion alert.

Quick checklist before escalating
  • Is Illustrator fully updated to the latest version?
  • Was the document saved before running the script?
  • Are all four artboards named SM, MD, L, XL in uppercase?
  • Was Illustrator fully restarted after installation?
  • Does the document have at least one artboard and one visible page item?

How it works technically

Palette Window

Opens as a ScriptUI palette window — stays floating over Illustrator and doesn't block interaction with your artwork.

Allow-list Security

Each button only runs scripts from an approved allow-list. Attempts to run any unlisted file are blocked with a security alert.

BridgeTalk Fallback

If a script can't run directly via $.evalFile, the panel automatically falls back to BridgeTalk to launch it — ensuring compatibility across all Illustrator versions.

Relative Paths

The panel finds all script files using paths relative to its own location in the AutoPerfect 2.0 folder — no hardcoded paths, works on any machine.

Important: The Smart App panel and all five script files must remain together in the same AutoPerfect 2.0 folder. If you move brix_smart_app.jsx out of the folder, it won't be able to find the other scripts.
